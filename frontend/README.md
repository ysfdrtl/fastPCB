# Frontend Katmani

Bu klasor, FastPCB platformunun istemci tarafini barindirir. Kullanici burada proje kesfi yapar, giris olur, proje yukler ve etkilesime girer.

## Alt Katman

- `FastPCB.Web`: React + Vite + TypeScript ile hazirlanan web arayuzu

## Genel Gorev

- Backend API'den veri cekmek
- Kullaniciya sayfa ve form deneyimi sunmak
- JWT tabanli oturum bilgisini tarayici tarafinda yonetmek
