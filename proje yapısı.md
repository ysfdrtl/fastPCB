# Proje Yapisi

## Kullanilan Teknolojiler

### Backend

- .NET Web API
- Entity Framework Core
- MySQL
- JWT Authentication

### Frontend

- React
- Vite
- TypeScript

## Mevcut Yapi Uygun mu?

Evet. Katmanli backend yapisi ve ayri frontend klasoru yeni proje fikri icin uygundur.

### Uygun Olanlar

- `API + Services + Data + Models` ayrimi
- MySQL veritabani
- JWT ile korunan endpointler
- Ayri web frontend uygulamasi

### Guncel Domain Karsiliklari

- `Project`
- `Comment`
- `ProjectLike`
- `Report`

## Sonuc

Mevcut proje yapisi artik PCB siparis sistemi gibi degil, topluluk tabanli bir PCB paylasim platformu gibi konumlanmistir.
